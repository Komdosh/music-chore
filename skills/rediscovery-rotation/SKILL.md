# Rediscovery Rotation

## Purpose

Help users fall back in love with their own library.

## Use When

- User feels bored with current listening habits.
- User wants overlooked tracks surfaced.

## Inputs

- `path` (optional)
- `max_tracks` (optional, default `12`)

## MCP Prompt

- `rediscovery-rotation`

## Expected Output

- a rediscovery queue
- short reason per track
- next rotation suggestions
