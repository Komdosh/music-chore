# Decision Duel

## Purpose

Choose between two listening directions.

## Use When

- User is torn between two moods, genres, or vibes.
- User wants a clear winner and ready queue.

## Inputs

- `path` (optional)
- `option_a` (required)
- `option_b` (required)
- `max_tracks_per_option` (optional, default `5`)

## MCP Prompt

- `decision-duel`

## Expected Output

- candidates for each option
- direct comparison
- winning direction with mini queue
