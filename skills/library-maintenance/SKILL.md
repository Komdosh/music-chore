# Library Maintenance

## Purpose

Keep the library clean and trustworthy with one compact workflow.

## Use When

- Metadata quality is inconsistent.
- Duplicates may exist.
- CUE state is unclear.

## MCP Prompts

- `library-health-check`
- `metadata-cleanup-guide`
- `duplicate-resolution`
- `cue-sheet-assistant`

## Expected Output

- prioritized issue list
- exact commands to run next
- safe cleanup plan with checkpoints
