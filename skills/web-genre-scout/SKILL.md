# Web Genre Scout

## Purpose

Discover external music in a specific genre while preserving user taste fit.

## Use When

- User asks for genre-focused expansion.
- User wants fewer misses and tighter relevance.

## Inputs

- `path` (optional)
- `genre` (required)
- `max_results` (optional, default `10`)

## MCP Prompt

- `web-genre-scout`

## Expected Output

- genre-fit summary
- ranked web recommendations with scores
- short "start here" sequence
