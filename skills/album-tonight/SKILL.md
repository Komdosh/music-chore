# Album Tonight

## Purpose

Choose one album session that fits the user's time window.

## Use When

- User wants a complete album experience.
- User has a defined time budget.

## Inputs

- `path` (optional)
- `available_minutes` (optional, default `60`)

## MCP Prompt

- `album-tonight`

## Expected Output

- one primary album
- listening plan for time constraints
- 2 backup albums
